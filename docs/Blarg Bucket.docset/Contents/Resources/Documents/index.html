Getting Started

Prerequisites

  1. Get CocoaPods $ sudo gem install cocoapods
  2. Get Jazzy for documentation $ sudo gem install jazzy
  3. Run pod install
  4. Run the tests

Generating the docs

$ jazzy -g https://github.com/luketheobscure/BlargBucket -m 'Blarg Bucket'
